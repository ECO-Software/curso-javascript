Tipos de datos

Los tipos de datos primitivos controlados por el operador typeof fuente: Mozilla

  • String: Cadena de texto
  • Number: Número
  • Boolean: Verdadero o falso
  • Undefined: No definido
  • Null: Nulo
  • Object: Objeto
